ยินดีต้อนรับเข้าสู่เว็ปไซต์เพื่อการศึกษาสำหรับการเรียนการสอน วิชาพื้นฐานการสร้างเว็ปไซต์

คำอธิบายรายวิชา ศึกษาและปฏิบัติเกี่ยวกับโครงสร้างของเว็ปไซต์ หลักหารออกแบบเว็ปไซต์และสร้างเว็ปไซต์ด้วยภาษา HTML 5 การตกแต่งข้อความ การแทรกรูปภาพหรือสื่อต่างๆ การสร้างการเชื่อมโยงในแบบต่างๆ การสร้างเฟรม การออกแบบและสร้างฟอร์ม การแทรก java Script การทดสอบการทำงานของเว็ปไซต์ การอัพโหลดเว็ปไซต์ไปยังเว็ปเซิร์ฟเวอร์
